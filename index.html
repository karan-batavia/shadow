Previously this plugin was developed by @johnrengelman (opens new window) and published under the ID com.github.johnrengelman.shadow (opens new window) before maintenance was transferred to the GradleUp organization (opens new window) to ensure future development, see #908 (opens new window).

If you are still using the old plugin ID in your build script, we recommend to switch to the new plugin ID com.gradleup.shadow (opens new window) and update to the latest version to receive all the latest bug fixes and improvements.
